# Hob-lang
## Getting Started
Requires GNU `gcc`, `cmake` and `llvm`
``` bash
mkdir .build
cd .build
cmake ../
cmake --build ./
```
## Syntax
look in `examples` :P
### To compile
```
cd examples
mkdir .build
hoblang build-exe examples/test.hob .build/test
.build/cstest
```
